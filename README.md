jquery.uncomment
================

jquery plugin to programatically uncomment HTML code

Given:  
`<div class="commented"><!-- A --></div>`  
`<div class="commented"><!-- B --></div>`

When:   
`$(".commented").uncomment();`  

Produces:   
`<div class="commented"> A </div>`   
`<div class="commented"> B </div>`
